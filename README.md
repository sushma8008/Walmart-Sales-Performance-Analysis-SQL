# Walmart Sales Performance Analysis using MySQL

## Project Overview

This project analyzes Walmart sales data using advanced MySQL queries to extract business insights related to sales growth, customer behavior, product profitability, payment methods, and customer segmentation.

The project demonstrates the use of SQL for solving real-world business problems through data analysis.

---

## Objectives

- Analyze branch-wise sales performance
- Identify profitable product lines
- Segment customers based on spending
- Detect unusual sales transactions
- Analyze payment method preferences
- Compare monthly sales by gender
- Identify repeat customers
- Find top customers
- Analyze weekday sales performance

---

## Key Insights

- Branch C achieved the highest sales growth.
- Electronic Accessories generated the highest profit.
- Customers were classified into High, Medium, and Low spending groups.
- Sales anomalies were identified for fraud detection.
- E-wallet and Cash were the preferred payment methods.
- Female customers contributed slightly higher sales.
- Weekend sales were significantly higher.
- Repeat customers were successfully identified.
